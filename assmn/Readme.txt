# ASSMN实现代码

## 环境及依赖
* Python 3.5.6
* Pytorch 1.1.0
* opencv-python 3.4.2
* numpy
* matplotlib
* scipy
* scikit-learn

## 操作步骤
1. 执行trainval.py进行模型训练，精度评定，并保存模型及精度评价结果
2. 执行infer.py进行模型预测
执行命令见trn.txt
